# ⚔️ 異世界健身大冒險 (Isekai Fitness Adventure)

> **「被女神召喚到異世界的你，唯一的武器就是... 強壯的肉體？！」**

這是一個結合 **RPG 角色扮演**、**AI 機器學習推薦**、**OpenAI 知識問答** 以及 **極限打字反應** 的文字冒險遊戲。玩家將扮演一位轉生到異世界的勇者，必須透過鍛鍊身體數值，擊敗佔據世界的五大「懶惰魔王」。

## 🌟 專案特色

* **🤖 AI 智能教練**：利用 `Random Forest` (隨機森林) 演算法，根據玩家隨機生成的身體數值（身高、體重、各部位肌力），推薦最適合的訓練菜單。
* **🧠 OpenAI 互動生成**：
    * **動作講解**：AI 化身熱血教練，即時講解健身動作的訓練主旨與注意事項。
    * **知識考核**：動態生成是非題，考驗玩家對健身動作的理解，答對才能獲得能力加成。
    * **傳說結局**：通關後，AI 會根據玩家的體重、訓練偏好及擊殺名單，撰寫獨一無二的勇者傳說。
* **⚡ 多重難度與存檔**：
    * 支援 4 個存檔欄位。
    * 設有 5 種遊戲難度（從「簡單」到「屌炸天」），決定打字戰鬥的極限反應時間（最快 0.1 秒/字）。

## 🎮 遊戲模式與玩法

### 1. ⚔️ 討伐魔王 (Battle Mode)
* **核心玩法**：極限打字反應遊戲。
* **機制**：
    * 遭遇隨機魔王，每個魔王有不同的血量（需成功次數）與代表單字長度。
    * 隨著魔王越強，單字長度越長。
* **難度係數**：根據開局選擇的難度（1-5），限制每個字母的輸入時間。
    * 例如：「屌炸天」難度下，每個字母只有 **0.1 秒** 反應時間。
* **逃跑**：戰鬥中輸入 `exit` 或 `q` 可戰術性撤退。

### 2. 🏋️ 道館訓練 (Gym Mode)
* **核心玩法**：AI 推薦與問答。
* **流程**：
    1.  系統分析玩家當前體力與數值，推薦 3-7 個訓練動作。
    2.  玩家輸入編號選擇訓練項目。
    3.  **AI 教練講解**該動作的主旨與注意事項。
    4.  **AI 出題考核**，玩家需回答 `T` (True) 或 `F` (False)。
    5.  答對後扣除體力，並增加該部位的訓練熟練度。

### 3. 🛌 休息 (Rest)
* 將所有部位的體力恢復至最大值。
* 自動儲存遊戲進度。

### 4. 📊 屬性與檔案 (Status & Archive)
* 查看玩家當前的身體數值（體重、各部位肌力上限）。
* 查看累積的各部位訓練次數。
* 查看已擊敗的魔王名冊與惡行介紹。

## 📂 檔案結構說明

本專案包含以下核心檔案，每個檔案負責不同的職責：

| 檔案名稱 | 說明 |
| :--- | :--- |
| **`main_game.py`** | **遊戲主程式入口**。負責遊戲迴圈、選單介面、存檔管理 (`pickle`)、以及整合所有支線功能。 |
| **`simulate.py`** | **玩家類別定義 (`Player`)**。負責隨機生成角色的屬性（性別、年紀、體重、各部位肌力值），以及狀態管理的邏輯。 |
| **`training_1.py`** | **AI 推薦核心**。包含 `Random Forest` 模型的訓練與預測邏輯。它會讀取 `csv` 資料，並根據玩家數值回傳適合的動作列表。 |
| **`mini_training_game.py`** | **道館訓練模組**。負責處理與 OpenAI API 的連線（講解、出題），以及訓練時的體力扣除邏輯。 |
| **`mini_typing_game.py`** | **打字戰鬥模組**。包含 `quick_reaction_game_strict` 函式，負責計時、判斷輸入是否正確，以及處理難度係數。 |
| **`exercise_movement.csv`** | **動作資料庫**。包含各種健身動作的名稱、訓練部位、消耗體力值與難度係數。 |
| **`processed_dataset.csv`** | **訓練資料集**。經過預處理的數據，用於訓練 Random Forest 模型。 |
| **`fitness_ai_model.pkl`** | **模型存檔**。程式自動生成的訓練好模型，加速下次載入時間。 |
| `saves/` | 自動生成的資料夾，存放玩家的存檔 (`.pkl`)。 |
